```cpp
int ex_gcd(int a,int b,int &x,int &y)
{
    if(!b)
    {
        x=1,y=0;
        return a;
    }
    int gcd = ex_gcd(b,a%b,y,x);
    y-=x*(a/b);
    return gcd;//返回gcd(a,b)
}

```
简单证明下：边界条件:$$gcd(a,0)=1*a-0*0=a$$
